This site maps every relevant place on Earth using open data from
**OpenStreetMap**. Every point you see on the [map](/map) was contributed by a
real person mapping the world.

## Where the data comes from

We query the OpenStreetMap **Overpass API** in tiles, deduplicate the results,
tag each point with its country, and store everything as a single static
GeoJSON file. No database, no tracking — just a fast map.

## How fresh is it?

A scheduled GitHub Action re-fetches the dataset **once a month**, commits the
updated GeoJSON, and triggers a new deployment automatically. The map is always
close to the live state of OpenStreetMap.

## Want a place added?

Add it to [OpenStreetMap](https://www.openstreetmap.org) — it will appear here
on the next monthly refresh.
